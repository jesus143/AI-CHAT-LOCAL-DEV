- **Frontend**: HTML/CSS/JavaScript

## Technical Highlights

During development, I encountered and solved several interesting challenges:

1. **Multiprocessing Issues**: Debugged semaphore leaks from sentence-transformers' parallel processing conflicting with Flask. Implemented environment-level fixes that eliminated crashes.

2. **Library Compatibility**: Migrated from ChromaDB to FAISS to resolve Python 3.13 and macOS compatibility issues, improving performance and reducing complexity.

3. **RAG Pipeline**: Designed an efficient chunking strategy (500-char chunks with 50-char overlap) that balances context preservation with search accuracy.
